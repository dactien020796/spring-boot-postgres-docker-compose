# Demo project for docker-compose with Spring Boot and PostgresSQL

Run steps:
1. Build the application: 
```
./gradlew clean build
```
2. Start the application:
```
docker-compose up
```

# Some more note:
- To shut-down, run the command:
```
docker-compose down
```
- To remove all the container and re-build, run the command:
```
docker-compose rm
```
